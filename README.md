# LRU Cache Implementation

## Description
A memory-efficient Least Recently Used (LRU) cache implementation. This project demonstrates core data structure principles, focusing on providing O(1) time complexity for both `get` and `put` operations.

## Key Functionalities
* Efficient cache eviction policy.
* Optimized memory management for stored data.
* O(1) performance for core cache operations.

## Technical Stack
* C

## Setup
1. Clone the repository: `git clone [repository-url]`
2. Compile the source code: `gcc main.c -o lru_cache`
3. Run the application: `./lru_cache`
